# Самостоятельное задание №1. Оформление README.md

## Описание лабораторной работы

Целью данной лабораторной работы является освоить принципы построения адаптивных макетов с использованием CSS Grid, научиться создавать сетки разной сложности и применять их в реальных проектах

---

## Теоретическая часть: Введение в CSS Grid
1. Подготовка HTML-файла (базовая структура документа, настройка метатегов, создание контейнеров для сетки)
2. Создание Grid-контейнера (активация display: grid, определение области сетки)
3. Определение столбцов и строк (использование grid-template-columns, grid-template-rows, единицы fr и repeat())
4. Управление отступами и промежутками (свойства gap, row-gap, column-gap)
5. Работа с именованными областями Grid (создание семантических областей, распределение элементов)
6. Выравнивание элементов (свойства align-items, justify-content, центрирование содержимого)

## Практическая часть
1. Создание файлов проекта (структура HTML и CSS, базовые стили)
2. Построение простой Grid-сетки (элементы в ячейках, настройка размеров)
3. Создание макета страницы (шапка, меню, контент, футер через Grid-области)
4. Разработка адаптивной галереи (карточки товаров или изображений в сетке)
5. Настройка отзывчивости (медиазапросы для изменения сетки на разных устройствах)
6.  Тестирование и отладка (проверка в браузере, исправление ошибок вёрстки)


## Сравнение Flexbox и CSS Grid

| Критерий         | Flexbox          | CSS Grid         |
|------------------|------------------|------------------|
| Тип компоновки   | Одномерная       | Двумерная        |
| Управление       | Строка или столбец | Строки и столбцы |
| Основное назначение | Выравнивание элементов | Построение макетов |
| Подходит для     | Компонентов      | Страниц и сеток  |

**Вывод:** Flexbox лучше подходит для **линейных структур**, Grid — для **табличных и сложных**.

### Типичные сценарии использования

| Задача                 | Рекомендуется     |
|------------------------|-------------------|
| Навигационное меню     | Flexbox           |
| Карточки товаров       | Flexbox + Grid    |
| Центрирование элемента | Flexbox           |
| Макет страницы         | Grid              |
| Галерея изображений    | Grid              |
| Дашборд                | Grid              |
